# Understanding Internet Addressing / Articulate StoryLine

An interactive e-learning module developed using **Articulate Storyline 360**, designed to educate users on the fundamental principles of internet addressing, including IPv4, IPv6, and subnetting.

## üöÄ Getting Started

### Prerequisites

To view or edit the `.story` source file, you will need:
* [Articulate Storyline 360](https://articulate.com/360/storyline)

### Installation & Usage

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/claymorenjoyer/internet_addressing_articulate_sl.git
    ```
2.  **Navigate to the directory:**
    ```bash
    cd internet_addressing_articulate_sl
    ```
3.  **View the module:** Open `index.html` in the SCORM folder using any modern web browser.
4.  **Edit the project:** Open the `.story` file in Articulate Storyline 360.
